Sempre fui apaixonado por tecnologia e a área de Ciência de Dados é a que mais me chama atenção. 

Estou sempre procurando evoluir e estudar, pois essa 
área não para! 🤣

Meus interesses variam demais 😆,

melhor a realidade. 🌎
